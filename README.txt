         x3py - C++ Plugin Framework
        http://github.com/rhcad/x3py/

x3py is designed as a plugin-framework that uses C++ and STL.
This framework can integrate with languages including Python by using SWIG.

For MSWindows installation notes, see INSTALL-win.txt .
For Linux, Mac OS X or other systems, see INSTALL-unix.txt .

FAQ and other hints may be found on the Wiki:
        http://github.com/rhcad/x3py/wiki
